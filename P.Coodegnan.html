E → Express.js (Backend framework)
R → React.js (Frontend framework)
N → Node.js (Backend runtime environment)

Backend lo Node.js + Express.js use chestam.
Core Concepts Node.js → JavaScript server-side runtime.
Express.js → Minimal backend framework.
Routing → /api/users, /api/products lanti endpoints create cheyyadam.
Middleware → Request/response processing
Environment Variables → dotenv for secure config. Database Connection
MongoDB → NoSQL DB, JSON-like documents store chestundi.
Mongoose → MongoDB ODM (Object Data Modeling) library.
Authentication & Authorization
JWT (JSON Web Token) Password hashing (bcrypt)
Role-based access (admin, user, etc.) Backend Tools
nodemon → Auto-restart server while developing.
postman → API testing.

SQL full Stack

SQL Full Stack Overview


Frontend → React.js / Angular / Vue.js (UI layer)
Backend → Node.js + Express.js / Python + Django / Java + Spring Boot (logic layer)
Database → MySQL / PostgreSQL (Relational DB)

Backend (Server-Side)


Backend lo Node.js + Express.js (or any backend framework) use chestam.
Core Concepts
Node.js → JavaScript server-side runtime.
Express.js → Minimal backend framework for APIs.
Routing → /api/users, /api/products lanti endpoints create cheyyadam.
Middleware → Request/response processing.
Environment Variables → dotenv for secure config.
Database Connection
MySQL → Most popular relational database.
PostgreSQL → Advanced relational DB with more features.
Knex.js / Sequelize → ORM (Object Relational Mapping) for Node.js.
SQL Queries → SELECT, INSERT, UPDATE, DELETE.
Joins → INNER JOIN, LEFT JOIN, RIGHT JOIN for combining data.
Authentication & Authorization
JWT (JSON Web Token) for stateless auth.
Password hashing (bcrypt).
Role-based access (admin, user, etc.).
Backend Tools
nodemon → Auto-restart server while developing.
postman → API testing.

Frontend (Client-Side)


Frontend lo React.js / Angular / Vue.js use chestam.
Core React Concepts
Components (Function & Class based)
JSX → JavaScript + HTML mix
Props & State → Data passing & state management
Hooks → useState, useEffect, useContext
React Router → Navigation between pages.
UI Frameworks / Styling
Tailwind CSS
Material UI (MUI)
Bootstrap
API Integration
Axios / Fetch for calling backend APIs.

Database Layer


MySQL → Structured schema with tables, columns, rows.
PostgreSQL → Advanced RDBMS, supports JSON + relational data.
Tables → Data organized into rows & columns.
Primary & Foreign Keys → For relationships.
CRUD Operations → Create, Read, Update, Delete.
Joins & Indexes → For optimized queries.

Tools & Extras


Version Control → Git & GitHub.
Package Management → npm or yarn.
API Testing → Postman / Thunder Client.
Deployment:
Backend → Render, Railway, Heroku.
Frontend → Vercel, Netlify.
Database → AWS RDS, Railway, or DigitalOcean Managed Databases.

Project Flow


Frontend (React.js / Angular) → UI design & components create cheyyadam.
Backend (Node.js + Express.js) → Business logic & API routes create cheyyadam.
Database (MySQL/PostgreSQL) → Tables create cheyyadam & data store cheyyadam.
API Communication → Frontend ↔ Backend via REST API.
Deployment → All services live server lo deploy cheyyadam.
